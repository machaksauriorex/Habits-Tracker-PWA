# Capturas de pantalla

Coloca aquí las capturas que se muestran en el README principal. Usa exactamente estos nombres:

- `hoy.png` — pantalla **Hoy**
- `hucha.png` — pantalla **Hucha**
- `estadisticas.png` — pantalla **Estadísticas**
- `ajustes.png` — pantalla **Ajustes**

Cómo hacerlas desde el móvil:
1. Abre la app (https://machaksauriorex.github.io/Habits-Tracker-PWA/) o la versión instalada.
2. Haz una captura de cada pestaña.
3. Sube las imágenes a esta carpeta con los nombres de arriba y haz commit.

Las imágenes son opcionales: el README funciona sin ellas (solo se verán los enlaces rotos hasta que las añadas).
